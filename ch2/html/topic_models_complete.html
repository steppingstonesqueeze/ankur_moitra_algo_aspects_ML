Recovery Algorithm (Pages 39-41)

Algorithm Steps:

1. Compute Gram Matrix G
Estimate P[w‚ÇÅ=j, w‚ÇÇ=j'] from document pairs
2. Find Anchor Words
Apply separable NMF to G
3. Solve Linear System
Recover P[t=i|w=j] for each topic-word pair
4. Apply Bayes Rule
Get P[w=j|t=i] = topic matrix A

Bayes Rule Recovery:

P[w=j|t=i] = P[t=i|w=j]P[w=j] / P[t=i]

where:

  • P[t=i|w=j] from linear system solution
  • P[w=j] estimated from document frequencies
  • P[t=i] = Œ£‚±º P[t=i|w=j]P[w=j]
